# homebrew-tap

Homebrew formulae for my open-source tools.

## Installation

```sh
brew tap mscheltienne/tap
```

## Available formulae

| Formula | Description |
|---------|-------------|
| [git-credit](https://github.com/mscheltienne/git-credit) | Precise per-author contribution stats that see through squash merges |

### git-credit

```sh
brew install mscheltienne/tap/git-credit
```
